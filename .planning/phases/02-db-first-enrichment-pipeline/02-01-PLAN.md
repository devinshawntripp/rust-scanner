<task type="auto">
  <name>Task 1: Create CircuitBreaker struct and add PG schema extensions</name>
  <files>src/vuln/circuit.rs, src/vuln/pg.rs, src/vuln/mod.rs</files>
  <action>
1. Create `src/vuln/circuit.rs` with a `CircuitBreaker` struct:
   - Fields: `failures: AtomicU32`, `threshold: u32`, `source_name: &'static str`
   - `pub fn new(source_name: &'static str, threshold: u32) -> Self` (NOT Arc-wrapped — caller wraps if needed)
   - `pub fn record_failure(&self)` — fetch_add(1), if failures == threshold emit progress event via `crate::utils::progress(&format!("{}.circuit_breaker.tripped", self.source_name), ...)`
   - `pub fn record_success(&self)` — store(0)
   - `pub fn is_open(&self) -> bool` — load >= threshold
   - `pub fn source_name(&self) -> &str`
   - Use `std::sync::atomic::{AtomicU32, Ordering}` with `Ordering::SeqCst`
   - Do NOT use static/OnceLock — this struct is instantiated per scan

2. In `src/vuln/mod.rs`, add `pub mod circuit;` and `pub use circuit::CircuitBreaker;` to the module declarations and re-exports.

3. In `src/vuln/pg.rs`, add to `pg_init_schema` batch_execute string:
   - `CREATE TABLE IF NOT EXISTS osv_batch_chunk_cache (chunk_digest TEXT PRIMARY KEY, payload JSONB NOT NULL, last_checked_at TIMESTAMPTZ NOT NULL);`
   - `CREATE TABLE IF NOT EXISTS epss_scores_cache (cve_id TEXT PRIMARY KEY, score REAL NOT NULL, percentile REAL NOT NULL, last_checked_at TIMESTAMPTZ NOT NULL);`
   - `CREATE TABLE IF NOT EXISTS kev_entries_cache (cve_id TEXT PRIMARY KEY, last_checked_at TIMESTAMPTZ NOT NULL);`

4. In `src/vuln/pg.rs`, add `compute_jittered_ttl_days(base_days: i64, jitter_days: i64) -> i64`:
   - Uses `rand::Rng` and `rand::thread_rng().gen_range(-jitter_days..=jitter_days)`
   - Returns `(base_days + jitter).max(1)`
   - Make it `pub(super)` visibility

5. In `src/vuln/pg.rs`, add PG helper functions:
   - `pub(super) fn pg_get_osv_batch_chunk(client: &mut PgClient, digest: &str, ttl_days: i64) -> Option<serde_json::Value>` — SELECT payload FROM osv_batch_chunk_cache WHERE chunk_digest = $1 AND last_checked_at > NOW() - make_interval(days => ttl_days)
   - `pub(super) fn pg_put_osv_batch_chunk(client: &mut PgClient, digest: &str, payload: &serde_json::Value)` — INSERT ON CONFLICT DO UPDATE SET payload, last_checked_at = NOW()
   - `pub(super) fn pg_put_epss_scores(client: &mut PgClient, scores: &[(String, f32, f32)])` — batch insert/upsert into epss_scores_cache
   - `pub(super) fn pg_get_epss_scores(client: &mut PgClient, cve_ids: &[&str], ttl_days: i64) -> std::collections::HashMap<String, (f32, f32)>` — SELECT cve_id, score, percentile WHERE cve_id = ANY($1) AND last_checked_at > threshold
   - `pub(super) fn pg_put_kev_entries(client: &mut PgClient, cve_ids: &[String])` — batch insert into kev_entries_cache ON CONFLICT DO NOTHING
   - `pub(super) fn pg_get_kev_entries(client: &mut PgClient, ttl_days: i64) -> std::collections::HashSet<String>` — SELECT cve_id FROM kev_entries_cache WHERE last_checked_at > threshold
   - Follow the existing pg_get_osv/pg_put_osv pattern in pg.rs for error handling (log warn, return None/empty on failure)
  </action>
